How it Works

Swipe right to add to your list. Your likes appear on your partner's "Partner's Picks" tab. Shows you both swiped right on appear in "Matches".

For cross-device sync (two phones), both devices must paste the same JSONBin key below. On one device, everything works automatically.

Local only
Get one at console.anthropic.com
Get one at themoviedb.org. Both v3 (short) and v4 (long eyJ...) formats work.
In your Google Sheet: File → Share → Publish to web → CSV. Paste that link, or the regular sheet URL (will auto-convert). Expected columns: Title, Type, Year, Rating.
Free at jsonbin.io. Account → API Keys → X-MASTER-KEY. Paste same key on both phones.
After setup click "Test Sync" — this auto-fills. Copy this ID to your partner's phone.
